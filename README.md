# Jose's Persona 3 Portfolio

A Persona 3 inspired personal portfolio built with React + Vite.

This project is based on the original work by **blairxu13**.
Full credit goes to them for the design, concept, and implementation.
Original repository: https://github.com/blairxu13/persona3-website

## Assets

You will need the assets from this folder:
https://drive.google.com/drive/folders/1_dVxHVTJO_YtMAo9ia7wnja1XY0nImr5?usp=drive_link

Place all files inside `src/assets/`.

## Fonts

All fonts are loaded from Google Fonts and require no local installation.

| Font | Weight(s) | Role |
|---|---|---|
| [Anton](https://fonts.google.com/specimen/Anton) | 400 | Main display — menu items, large headings |
| [Bebas Neue](https://fonts.google.com/specimen/Bebas+Neue) | 400 | UI chrome — stat labels, nav elements, bar text |
| [Barlow Condensed](https://fonts.google.com/specimen/Barlow+Condensed) | 400, 700, 700 italic | Accent / condensed text |
| [Montserrat](https://fonts.google.com/specimen/Montserrat) | 300 | Body text — reveal panels, descriptions |

## Getting Started

```bash
npm install
npm run dev
```
